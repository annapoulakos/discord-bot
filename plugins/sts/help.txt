The Slay the Spire plugin returns information about Slay the Spire cards.

`/slay CARDNAME`
Returns the best match for CARDNAME

`/slay --info CARDNAME`
Returns information about the specified CARDNAME. Aliases: `-i`

`/slay --help`
Displays this help. Aliases: `-h`

`/slay --version`
Displays the current version and changelog. Aliases `-v`
